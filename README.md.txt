# Fraud Detection System

Machine Learning project that detects fraudulent transactions using classification algorithms.

## Features

- Fraud transaction prediction
- Flask web application
- Improved recall score
- Handles imbalanced datasets

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

## Project Structure

```bash
fraud-detection-system/
```

## Run Application

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```

## Machine Learning

- Classification Algorithm
- SMOTE balancing
- Recall optimization
